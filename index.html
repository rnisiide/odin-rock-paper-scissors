Do you want to play a game?

This is a simple Rock, Paper and Scissors game. For now, it is still played in the Console Log:

  1. Press F12 to open Dev Tools, and go to CONSOLE.
  2. Type in the console the function: game()
  3. A prompt will appear to choose one of the following values:
    • Rock
    • Paper
    • Scissors
  4. There will be 5 rounds. You can choose different values each time.
  5. The values are case iNsEnSiTivE, so you don't need to worry about CAps LoCk. However, you have to spell it properly, or it will return an INVALID OPTION.
  6. After 5 rounds the result will show in the end!!
  7. You can start a new game with the function: game()

Good luck!!
